# INDIANA SHONES AND THE HIDDEN TRESURE IN CAVE

## Requirements

- pasmo
    https://pasmo.speccy.org/

- openmsx
    https://openmsx.org/

- basic-bios

<br>

## How to make and execute

1. Compile asm source
```
cd source
pasmo --msx shones1.asm program.bin
```

2. Execute openmsx (use DiskBasic-bios)
```
openmsx -diska .
```
